# Specification

## Summary
**Goal:** Remove the 'institution' course from the application.

**Planned changes:**
- Remove 'institution' course from backend courses map
- Remove 'institution' course from migration module
- Remove 'institution' course image mapping from frontend utility
- Verify 'institution' course no longer appears in any course listings

**User-visible outcome:** The institutional course will no longer be visible or available for enrollment anywhere in the application. Only the basic, advanced, super, and meditation courses remain.
