# Specification

## Summary
**Goal:** Add a meditation course for 18+ age group and update all existing course descriptions to include duration and daily class hours.

**Planned changes:**
- Add new meditation course with 18+ age requirement to course catalog
- Update Basic Midbrain Activation course description with duration: 3 days, 1 hour online classes daily
- Update Advance Midbrain Activation course description with duration: 15 days, 1+ hours online classes daily
- Update Institution course description with duration: 1 month, 1+ hours online classes daily
- Display all updated course descriptions with duration details on public homepage, course list, and student dashboard
- Assign unique image to meditation course and display across all views

**User-visible outcome:** Users will see a new meditation course available for enrollment (18+ only) and all courses will display clear duration and daily class hour information throughout the application.
